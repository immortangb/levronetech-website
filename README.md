# Levrone Technologies Website

A professional IT company website built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- **Responsive Design** - Works on all devices
- **Modern UI/UX** - Clean, professional design
- **Service Showcase** - 6 services with expandable details
- **Product Catalog** - 8 laptops with brand filtering
- **Contact Form** - Functional contact form
- **Student Discount** - 10% discount for UFS students

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React (icons)
- Sonner (toast notifications)

## Deployment to Vercel

### Step 1: Push to GitHub
1. Create a new repository on GitHub
2. Upload all files from this ZIP
3. Commit the changes

### Step 2: Deploy on Vercel
1. Go to [vercel.com](https://vercel.com) and sign in
2. Click "Add New" → "Project"
3. Import your GitHub repository
4. Vercel will auto-detect settings
5. Click "Deploy"

### Step 3: Custom Domain (Optional)
1. In Vercel dashboard, go to Settings → Domains
2. Add your domain: `levronetech.co.za`
3. Follow DNS instructions

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
